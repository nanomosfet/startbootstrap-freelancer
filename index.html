Surf Spot Map Viewer

This single page application provides the user with a list of surf spots provided by api.spitcast.com.

This project is built using Knockout, spitcast api and the Google Maps API page

It is also using WebPack and babel in its build pipeline in order to take advantage of ES-2107 Javascript

The example hosts the application with flask as its server run inside of a Docker container

Custom Form Creator

This single page application provides the user with a way to customize a form.

This project is built using React, React Drag and Drop.

It uses the create-react-app tool as the build environment

The example hosts the application with flask as its server run inside of a Docker container

It currently is in progress. Some features that will come out are the ability to switch view to how the form would look for a user

The ability to share a form and then view the data that comes back from the user.

SSL Grade Checker

SSL grade checker is a way for a someone to track the SSL configuration of their pages.

Currently it does not recheck unless you ask and there is no user login.

This application uses vanilla js written with MVC in mind.

On the backend I used flask and sqlalchemy for the web client and database

It also takes the ssllabs API to analyze the SSL configurations of a page

The example hosts the application with flask as its server run inside of a Docker container

Multi User Blog

The multi user blog is an example of a blog for multiple users.

It is hosted from google app engine.

It uses the standard Google Cloud SDK for Python and Jinja2 for templating.

The site supports multiple users. Users have the ability to post blogs with a photo, a title, and some content. Users can also like other people Blogs. Go add a blog!

It is responsive and works on all device sizes
